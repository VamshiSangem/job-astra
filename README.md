# Job Astra 💼✨

**Job Astra** is an AI-powered assistant that helps job seekers automate resume tailoring, job tracking, and "Easy Apply" submissions.

---

### 🚀 Features

- 📝 Resume rewriting using OpenAI
- 🔍 Job scraping from LinkedIn, Indeed, and more
- 🤖 Telegram + Email summary integrations
- 📊 Google Sheets & Drive logs
- 🔁 Built with modularity for job search automation

---

### 🛠️ Tech Stack

- Python
- OpenAI API
- Google Sheets API
- Make.com
- Telegram Bot API
- UI.Vision RPA

---

### 🌱 Vision

Empowering immigrant students and job seekers to apply confidently – with clarity, ethics, and automation.
